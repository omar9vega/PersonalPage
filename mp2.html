1. (0:00) List all the galleries (including name and descriptions).
2. (0:14) List all the images and the number of images in a gallery (including title and link).
3. (0:26) List the details of a given image (This function should be a link from the result of function 2. It should show the picture, the artist name, and all the details of the artwork.)
4. (0:45) List the details of an artist. (This function should be a link from the result of function 3.)
5. (1:04) Create a new gallery.
6. (1:43) Create a new artist.
7. (2:10) Add a new image to a gallery. To simplify, you can just input the link of the picture instead of actually uploading it. This function should also input the artist_id, and input all the details of the image.
8. (3:33) Delete an image from the gallery. You should also delete the details of the image but not the artist.
9. (4:00) Modify the details of an image (including title and link).
10.(4:38) Modify the details of an artist.
11.(5:23) Modify the title and description of a gallery.
12.(5:58) Find the images by type (“Find” means to list all the results.)
13.(6:50) Find the images by a given range of creation year.
14.(6:13) Find the images by artist name.
15.(6:39) Find the images by location.
16.(7:09) Find the artists by country.
17.(7:30) Find the artists by birth year.
